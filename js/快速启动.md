# 🚀 快速启动指南

## 立即开始使用

### 方法一：直接打开（最简单）
1. 进入 `js` 目录
2. 双击 `index.html` 文件
3. 开始使用！

### 方法二：使用本地服务器（推荐）

**Windows 用户（使用 Python）：**
```powershell
cd js
python -m http.server 8000
```

**Windows 用户（使用 Node.js）：**
```powershell
cd js
npx http-server -p 8000
```

然后在浏览器中打开：`http://localhost:8000`

## 🎯 使用示例

### 示例 1：解码 .gil 文件
1. 打开 `index.html`
2. 选择 "📥 解码模式"
3. 上传 `.gil` 文件
4. 点击 "🔄 开始转换"
5. 自动下载 JSON 文件

### 示例 2：编码 JSON 文件
1. 打开 `index.html`
2. 选择 "📤 编码模式"
3. 上传 `.json` 文件
4. 点击 "🔄 开始转换"
5. 自动下载二进制文件

## 📝 注意事项

### ⚠️ 重要提示
- **隐私安全**：所有处理都在浏览器本地完成，文件不会上传到服务器
- **文件大小**：建议处理的文件不超过 100MB
- **浏览器支持**：需要支持 ES6+ 的现代浏览器（Chrome、Firefox、Edge、Safari）

### 🔧 dtype 文件
- 解码时可选择提供 dtype CSV 文件
- 如不提供，程序会尝试从 `../dtype/` 目录加载内置 dtype
- 也可以完全不使用 dtype，让程序自动分析

## 🐛 问题排查

### 问题：文件转换失败
**解决方案：**
1. 确认文件格式正确
2. 尝试提供对应的 dtype 文件
3. 查看浏览器控制台（F12）的错误信息

### 问题：无法加载 dtype 文件
**解决方案：**
1. 手动选择 dtype CSV 文件
2. 或者将 dtype 文件放在正确的位置（`../dtype/`）

### 问题：页面显示异常
**解决方案：**
1. 清除浏览器缓存
2. 使用最新版本的浏览器
3. 尝试使用本地服务器运行
